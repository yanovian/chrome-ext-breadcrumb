# Changelog

All notable user-facing changes to Breadcrumb.

## [3.2.0] — Unreleased

- Available on the [Chrome Web Store](https://chromewebstore.google.com/detail/breadcrumb/medkklaaakhdpgcoaghblbnleeandifi).
- New [getting-started tutorial](./_doc/tutorial.md).
- Store listing descriptions in 40+ languages.

## [3.1.0] — 2026-07-03

- Extension UI translated into 40+ languages.

## [3.0.0] — 2026-07-03

- Fully offline AI search — the embedding model ships inside the extension, so Breadcrumb makes no network requests at runtime.

## [2.0.0] — 2026-07-03

- Offline AI search — the AI runtime is bundled in the extension package.

## [1.1.0] — 2026-07-03

- First release.
- Highlight text and save with right-click.
- Search saved notes by keyword or meaning (on-device AI).
- Learning timeline and auto topic tags.
- Export or delete all your data anytime.
